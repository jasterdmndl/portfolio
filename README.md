# 🌍 Developer Universe Portfolio

> **Building software that grows with every challenge.**

A modern, interactive portfolio built to showcase my journey as a software engineer. Rather than being a traditional portfolio, this project is designed as a living software product where every project becomes part of a growing developer universe.

The centerpiece of the experience is an interactive **Low-Poly Developer Planet**, representing my continuous growth, curiosity, and passion for building meaningful software.

---

## ✨ Features

- 🌍 Interactive 3D Developer Planet
- 🚀 Modern Hero Experience
- 📱 Fully Responsive Design
- 🎨 Premium Dark Theme
- ⚡ Smooth Animations
- 📂 Dynamic Project Showcase
- 🔗 GitHub Integration
- 📄 Downloadable Resume
- 📬 Contact Section
- ♿ Accessibility Focused
- ⚙️ Performance Optimized
